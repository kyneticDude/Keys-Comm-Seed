Assets Based Community Development Model

Keys Communications uses the ABCD Model, which fosters community affinity and loyalty to brands/products and services. ABCD = Assets Based Community Development Model, this is using existing Township resources, including people, knowledge and structures for mutual benefits between the advertiser and the community.
